# Telecom-Churn-Analysis
Predicting customer churn in the telecommunications industry using SQL to identify at-risk users and improve retention strategies.
## SQL Analysis Portfolio
This project uses SQL to analyze customer retention for a telecommunications provider.

### Key Analysis Components:
- Churn Rate Calculation:** Using Common Table Expressions (CTEs).
- Cohort Analysis:** Grouping customers by tenure to see when they typically drop off.
- Risk Segmentation:** Categorizing customers into High, Medium, and Low risk based on service usage.
